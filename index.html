#1.打开IDEA 2024，选择“Projects/项目”，点击“New Project/新建项目”，然后选择“Java”，填写项目名称及项目存放路径后，点击“create”创建。

#2.创建完成后为项目添加Web模块: 方式有两种

方式一: 双击 shift 搜索 Add Frameworks Support 、添加框架支持( 汉化版搜这个), 勾选 web Application

方式二: 在项目根文件夹上右键 选择 打开模块设置/Project Structure 或按 F4 选择Faces 再选择web模块
